# Flight Delays and Cancellations Analysis ✈️

Analysis of flight delays and cancellations in Australia using Python, Pandas and Seaborn.

## Tools Used
- Python 🐍
- Pandas 🐼
- Matplotlib 📊
- Seaborn 🎨

## Dataset
Australian flight data including routes, airlines, cancellations and delays.

## Key Findings
- Most flights have a cancellation rate below 10% (right-skewed distribution)
- Regional Express has the highest median cancellation rate (10.4%)
- QantasLink is the most reliable airline with a median of 0%
- Cancellation rates peak in June and July (Australian winter)
- Melbourne-Sydney is the most affected route

## Visualizations
- Histogram of cancellation rates
- Boxplot by airline
- Heatmap by route and month
